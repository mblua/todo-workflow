# TO-DOs Directory - Agent Instructions

This directory contains pending tasks for the todo-workflow repository itself.

**Note:** This is the workflow being used to develop the workflow template. Meta, right?

For the complete workflow documentation, see `template/TO-DOs/AGENTS.md`.

## Quick Reference

- **File naming:** `###-short-description.md`
- **Next ID:** Read from `NEXT_ID.txt`
- **Plan files:** `###_PLAN_<NAME>.md`
- **Completed tasks:** Move to `DONE/`

## Workflow Steps

1. Create TODO (wait for approval)
2. Generate Plan (wait for approval)
3. Review Plan in plan mode (wait for approval)
4. Analyze Improvements (wait for approval)
5. Generate Tests (auto-run, must fail)
6. Implement (wait for "implement")
7. Run Tests (must pass)
8. Complete Task (wait for "commit")
9. Commit Changes

See `template/TO-DOs/AGENTS.md` for full details.
